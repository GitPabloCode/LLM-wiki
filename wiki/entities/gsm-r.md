# GSM-R

**Source:** [[subset26-1]] | **Date:** 2026-05-12 | **Type:** entity

GSM-R (Global System for Mobile Communications - Railway) is the radio communication network used for bi-directional exchange of messages between on-board subsystems and RBC or Radio Infill Units in ERTMS/ETCS. [subset26-1 ¶110]

## Trackside Role

The GSM-R radio communication network provides the medium for bi-directional data exchange between:
- On-board subsystems and RBCs [subset26-1 ¶110]
- On-board subsystems and Radio Infill Units [subset26-1 ¶110]

## On-Board Role

The GSM-R on-board radio system is used for the same bi-directional exchange of messages. [subset26-1 ¶138]

## Relationship to Euroradio

GSM-R is the underlying communication network; Euroradio defines the application protocol and safe communication layer that operates over GSM-R.
