# Euroloop

**Source:** [[subset40]] | **Date:** 2025-07-17 | **Type:** entity

Euroloop is a loop-based transmission system used in ERTMS/ETCS as an alternative to spot transmission via Eurobalises, enabling continuous or semi-continuous communication with passing trains. [subset40 ¶94][subset40 ¶96]

## Installation Rules

All installation rules given in the FFFIS Euroloop (SUBSET-044) must be respected: [subset40 ¶96]

- Section 6.1.3: Arrangements
- Section 6.10: Trackside Installation Rules
- Section 6.11: Specific Electrical Requirements
- Section 6.13: EMC Requirements
- Section 7.8: Installation constraints for the Antenna Unit

Balise telegrams are transmitted by individual balises, while messages for Euroloops (and balise groups) are transmitted differently — a telegram comes from one balise, while a message comes from a balise group, loop, or radio. [subset40 ¶54]
