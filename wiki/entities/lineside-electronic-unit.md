# Lineside Electronic Unit (LEU)

**Source:** [[subset26-1]] | **Date:** 2026-05-12 | **Type:** entity

A Lineside Electronic Unit (LEU) is an electronic device that generates telegrams to be sent by balises, based on information received from external trackside systems. [subset26-1 ¶108]

## Function

The LEU enables Eurobalises to transmit variable (changeable) messages rather than fixed messages. When a balise is connected to a LEU, the message content can be changed dynamically based on signalling conditions (e.g., signal aspect changes). [subset26-1 ¶105]
